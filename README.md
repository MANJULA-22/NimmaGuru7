# Nimma Guru – Mentor & Student Connect Platform

## Project Title

**Nimma Guru – AI Powered Mentor & Student Learning Platform**

---

# Problem Statement

Students often struggle to find proper mentorship, guidance, skill development opportunities, and communication with experienced mentors in one centralized platform. Existing systems lack personalized mentor discovery, multilingual support, appreciation systems, and easy interaction between students and mentors.

Nimma Guru solves this problem by providing a mobile platform where students can connect with mentors (gurus), explore skills, attend sessions, communicate with mentors, and receive guidance in an easy and user-friendly way.

---

# Project Overview

Nimma Guru is an Android application developed using Jetpack Compose and Firebase. The app helps students discover mentors based on categories, skills, and interests. Mentors can create profiles, manage sessions, and interact with students.

The application focuses on:

* Mentor discovery
* Student engagement
* Multilingual support
* Community interaction
* Real-time data management
* Simple and responsive UI

# Installation Steps

## Prerequisites

Before running the project, make sure you have:

* Android Studio installed
* Android SDK configured
* Firebase project setup
* Internet connection

## Clone Repository

```bash
git clone https://github.com/your-username/nimma-guru.git
```

## Open Project

1. Open Android Studio
2. Click on **Open Project**
3. Select the cloned project folder

## Firebase Setup

1. Create a Firebase project
2. Add Android app to Firebase
3. Download `google-services.json`
4. Place it inside:

```bash
app/google-services.json
```

## Sync Project

* Click **Sync Gradle**
* Wait until dependencies are downloaded

---

# Run Command

## Using Android Studio

1. Connect Android device or start emulator
2. Click the **Run ▶** button

## Using Terminal

```bash
./gradlew assembleDebug
```
